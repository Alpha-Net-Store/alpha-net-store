# 🌐 متجر Alpha Net

متجر تطبيقات عربي حديث لعرض وبيع التطبيقات بسهولة.

## ✨ الميزات

- 📱 عرض التطبيقات بتصميم عصري
- 🔐 لوحة نشر آمنة بكلمة سر
- 💳 نظام دفع متعدد (شام كاش، واتساب، مكتب حوالة)
- 📲 مسح QR Code للدفع
- 💰 نظام تتبع المعاملات
- 🔍 تصفية التطبيقات حسب النوع
- 📊 Firebase Realtime Database

## 🚀 البدء السريع

1. **فتح الملف:**
   ```
   فقط افتح الملف HTML مباشرة في المتصفح
   ```

2. **إعداد البيانات الشخصية (السطر 34-39):**
   ```javascript
   window.MY_CONFIG = {
     whatsapp: '0992972576',
     shamcash: 'f9a481638647fc6e96fa69f0b1d896a',
     publisherPassword: '50829355AMLPHA'
   };
   ```

3. **إضافة تطبيق:**
   - اضغط ⚙️ لوحة النشر
   - أدخل كلمة السر
   - ملء البيانات والنشر

## 💻 التكنولوجيا

- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **Backend:** Firebase Firestore
- **QR Code:** QRCode.js
- **RTL Support:** دعم كامل للعربية

## 🔒 الأمان

✅ HTML Escaping ضد XSS attacks
✅ URL Validation
✅ Rate Limiting على عمليات الدخول
✅ Input Validation
✅ Firebase Security Rules

## ⚠️ نقاط الأمان المهمة

1. **غيّر كلمة السر الضعيفة** من `alpha2024` إلى رمز قوي
2. **استخدم Firebase Security Rules** لحماية البيانات
3. **لا تضع بيانات حساسة** في الكود الظاهر

## 📝 الملفات الرئيسية

## 🔧 التخصيص

### تغيير الألوان:
ابحث عن `:root` في CSS وعدّل المتغيرات

### تغيير Firebase:
عدّل `firebaseConfig` في السطر 22

### تغيير لوحة التصميم:
عدّل الأيقونات والألوان في `MY_CONFIG`
